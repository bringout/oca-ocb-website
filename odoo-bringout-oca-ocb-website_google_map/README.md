# Google Maps


Show your company address/partner address on Google Maps. Configure an API key in the Website settings.
    

## Installation

```bash
pip install odoo-bringout-oca-ocb-website_google_map
```

## Dependencies

This addon depends on:
- base_geolocalize
- website_partner
