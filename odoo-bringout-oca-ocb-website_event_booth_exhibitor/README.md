# Booths/Exhibitors Bridge


Automatically create a sponsor when renting a booth.
    

## Installation

```bash
pip install odoo-bringout-oca-ocb-website_event_booth_exhibitor
```

## Dependencies

This addon depends on:
- website_event_exhibitor
- website_event_booth
