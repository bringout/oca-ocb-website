# Send SMS to Visitor with leads

Allows to send sms to website visitor if the visitor is linked to a lead.

## Installation

```bash
pip install odoo-bringout-oca-ocb-website_crm_sms
```

## Dependencies

This addon depends on:
- website_sms
- crm
