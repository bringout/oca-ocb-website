# Customer References


Publish your customers as business references on your website to attract new potential prospects.
    

## Installation

```bash
pip install odoo-bringout-oca-ocb-website_customer
```

## Dependencies

This addon depends on:
- website_crm_partner_assign
- website_partner
- website_google_map
